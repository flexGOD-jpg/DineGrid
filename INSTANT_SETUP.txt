🔥 OPTION 2: PERMANENT STORAGE (5 MINUTES - RECOMMENDED!)
═════════════════════════════════════════════════════════════════════════════════

STEP 1: Get MongoDB Atlas (1 minute)
────────────────────────────────────────

1. Open: https://www.mongodb.com/cloud/atlas
2. Click "Create an Account"
3. Enter email/password (FREE, no credit card!)
4. Verify your email


STEP 2: Create Cluster (2 minutes)
────────────────────────────────────────

1. Login to MongoDB Atlas
2. Click "Create" → "Build a Cluster"
3. Choose "FREE" (M0)
4. Select region close to you
5. Cluster name: "dinegrid"
6. Click "Create"
7. Wait 2-3 minutes...


STEP 3: Setup Security (1 minute)
────────────────────────────────────────

In MongoDB Atlas:

A. Create Database User:
   • Go to "Security" → "Database Access"
   • Click "Add New Database User"
   • Username: dinegrid
   • Password: Create a strong one (write it down!)
   • Click "Add User"

B. Allow Network Access:
   • Go to "Security" → "Network Access"
   • Click "Add IP Address"
   • Click "Allow Access from Anywhere"
   • Click "Confirm"


STEP 4: Get Connection String (1 minute)
────────────────────────────────────────

1. Click "Clusters" (left menu)
2. Click "Connect" on your cluster
3. Choose "Drivers"
4. Copy the connection string (looks like):

   mongodb+srv://dinegrid:PASSWORD@cluster0.mongodb.net/dinegrid?retryWrites=true&w=majority

5. Replace PASSWORD with your database password!


STEP 5: Configure Your Backend (1 minute)
────────────────────────────────────────

Create a file: c:\Users\hazar\Desktop\mvp\setup.env

Paste this (replace YOUR_PASSWORD and YOUR_CONNECTION_STRING):

────────────────────────────────────────────────────────────────

MONGODB_URI=mongodb+srv://dinegrid:YOUR_PASSWORD@cluster0.mongodb.net/dinegrid?retryWrites=true&w=majority
ADMIN_USERNAME=admin
ADMIN_PASSWORD=SUBHAMDAXXX
PORT=5000

────────────────────────────────────────────────────────────────

Save the file!


STEP 6: Start Your Backend (30 seconds)
────────────────────────────────────────

In PowerShell:

   cd c:\Users\hazar\Desktop\mvp
   node server-db.js

You should see:

   ✅ ================================
   🚀 MongoDB Connected Successfully!
